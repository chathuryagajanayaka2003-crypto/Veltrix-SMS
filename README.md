# 🛒 Veltrix Supermarket Management System (SMS)

Veltrix SMS is a modern, scalable Supermarket Management System designed for real-world retail operations. It provides a complete solution for managing sales, inventory, customers, suppliers, and analytics in one platform.

## 🚀 Features

- 🧾 Point of Sale (POS) System
- 📦 Inventory & Stock Management
- 🛍️ Product & Category Management
- 👥 Customer Management
- 🚚 Supplier Management
- 💰 Sales & Purchase Tracking
- 📊 Real-time Analytics Dashboard
- 📈 Reports & Insights
- 🎟️ Discounts & Promotions System
- 🔔 Notifications System
- 🔐 Authentication & Role-based Access Control

## 🏗️ Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Zustand (State Management)
- React Query
- Socket.IO (Real-time updates)
- Node.js Backend (planned)
- PostgreSQL / MongoDB (planned)

## 🎯 Project Goal

To build an enterprise-level supermarket management system that can be used in real retail businesses with scalable architecture and modern UI/UX.
